## Cloudflare Routing

`_routes.json` declares which paths Cloudflare routes to the Functions layer vs. static SPA:

```json
"include": ["/profile/*", "/vport/*", "/m/*", "/p/*", ...]
```

All included paths are served as SPA (`index.html`). React Router then decides public vs. protected.

`_redirects` has one rule: `/* /index.html 200` â€” SPA fallback for everything not handled by a Cloudflare Function. Changing `_redirects` or `_routes.json` affects which paths receive the SPA shell â€” but React Router still controls whether an unauthenticated visitor can see the route content.

## TRAZE Integration Notes
