## What Was Observed

### Feed Pipeline (per page load)
- 7 DAL calls reading ~10 tables
- Up to 30 uncached table reads for 3-page pagination
- `readActorsBundle` + `hydrateActorsByIds` = confirmed duplicate read
- Block and follow rows re-fetched every page (session-stable data)

### Notification Badges
- `useNotiCount`: 15s TTL cache + 60s polling + realtime subscription
- `useUnreadBadge`: 10s TTL cache + 20s polling + realtime subscription
- Both consumed only in `BottomNavBar.jsx`
- BottomNavBar unmounts/remounts on route changes → subscription churn
