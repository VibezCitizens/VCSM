## Key Findings

1. **No `_headers` file** — zero security headers in production. CRITICAL gap.
2. **All 6 Pages Functions** return no security headers. They bypass `_headers` and need explicit header injection.
3. **`OnemoredaysAd.jsx` iframe** uses `sandbox="allow-same-origin allow-scripts ..."` — a known sandbox escape combination flagged by the HTML spec. HIGH severity.
4. **No Referrer-Policy** — `?invite_code=` in registration URLs leakable via Referer on external links.
5. **No X-Frame-Options** — clickjacking possible on all routes including `/register` and invite flows.
6. **og:image URL trust** — `card.js` and `menu.js` accept any HTTPS URL for user logo/avatar in OG meta tags without CDN allowlist validation.
7. **iosProdDebugger window globals** — `window.vcGetIosProdLog` exposed in production with internal session state.
