### Why This Way

`can_manage_domain` is SECURITY DEFINER so it can read `moderation.moderators` bypassing RLS regardless of who calls it. `auth.uid()` is JWT-based and resolves the session user correctly inside SECURITY DEFINER functions. The `is_current_user_moderator()` wrapper follows the same pattern as the old `learning.is_current_user_platform_admin()`, making the DAL change a clean one-line schema+name swap.
