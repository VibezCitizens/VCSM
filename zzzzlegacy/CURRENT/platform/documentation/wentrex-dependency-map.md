## Supabase Client Architecture

**Singleton client (HMR-safe):**
- File: `features/services/supabase/supabaseClient.js`
- Auth persistence: localStorage key `sb-auth-main`
- Debug mode: `VITE_DEBUG_SUPABASE='1'` or localStorage flag
- Expose to window: `VITE_EXPOSE_SB_CLIENT='1'` (dev only — see security report)

All learning DAL files use:
```javascript
supabase.schema("learning").from("table_name")
```

No direct `supabase.from()` without schema prefix in learning layer (exception: public.profiles for user auth joins).
