## Multi-Tenancy Model

```
Realm (school/institution) ← URL slug e.g. /learning/springfield-high
  └─ Organization
       └─ Course Term
            └─ Course
                 └─ Module
                      └─ Lesson
                           └─ Assignment
                                └─ Submission
                                     └─ Grade
```

Every query must be scoped to a Realm. RLS enforces tenant isolation at DB level.

## User Roles

| Role | Access |
|------|--------|
| Student | Takes courses, submits assignments, tracks progress |
| Teacher/Staff | Delivers content, grades, creates rubrics |
| Parent/Observer | Read-only view of linked student progress |
| Admin | Manages organizations, rosters, platform access |
| Super Admin | Creates and monitors all school workspaces |
