# Planning ‚Äî April / 06 / Sequence 01

Migrate all reporting, blocking, and reporter-local hide flows from old VC-specific moderation tables to the new neutral `moderation.*` schema.

## Old Tables (remove all writes)
- `vc.reports`
- `vc.report_events`
- `vc.moderation_actions`
- `vc.user_blocks`

## New Tables (replace with)
- `moderation.reports`
- `moderation.report_events`
- `moderation.actions`
- `moderation.blocks`
- `moderation.block_events`

## Flows to Migrate

### 1. Report a Conversation
- reporter_domain='vc', target_domain='chat', target_type='conversation'
- Insert moderation.reports + moderation.report_events
- Keep spam‚Üíinbox side effect

### 2. Report a Post
- reporter_domain='vc', target_domain='vc', target_type='post'
- Insert moderation.reports + moderation.report_events
- Reporter-local hide ‚Üí moderation.actions (action_type='hide', reason='user_reported_post')

### 3. Block a User
- blocker_domain='vc', blocked_domain='vc'
- Insert moderation.blocks (status='active') + moderation.block_events (event_type='blocked')
- Keep side effects: delete vc.actor_follows + vc.friend_ranks both directions
- Idempotent (no error on duplicate)

### 4. Unblock a User
- Update moderation.blocks SET status='released', released_at=now()
- Insert moderation.block_events (event_type='unblocked')
- Do NOT restore follows/friends

## Column Mapping for moderation.reports
- reporter_domain, reporter_actor_id
- target_domain, target_type, target_id
- parent_target_domain, parent_target_type, parent_target_id (all null for now)
- reason_code, reason_text, status='open', priority=3
- dedupe_key, meta

## Column Mapping for moderation.actions
- report_id (link to report when available)
- actor_domain='vc', actor_id
- target_domain, target_type, target_id
- action_type='hide', reason, meta

## Column Mapping for moderation.blocks
- blocker_domain='vc', blocker_actor_id
- blocked_domain='vc', blocked_actor_id
- status='active'/'released', reason, released_at, meta

## Column Mapping for moderation.block_events
- blocker_domain, blocker_actor_id, blocked_domain, blocked_actor_id
- event_type='blocked'/'unblocked'
- actor_domain='vc', actor_id, reason, meta
