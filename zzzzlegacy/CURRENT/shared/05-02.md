==================================================
WHAT "BATCH NOTIFICATION" MEANS
==================================================

Batch notifications are systems that group multiple events before notifying the user.

Examples:

Instead of:
User receives 5 notifications for 5 messages.

The system sends:
"You have 5 new messages from Alex."

Possible batching mechanisms include:

- server-side batching
- notification aggregation
- delayed push notifications
- outbox/event queues
- debounce windows
- digest systems
