# Planning — April / 05 / Sequence 17

Remove all legacy `vc.*` DAL files from VCSM's chat feature. VCSM chat must use the shared chat engine (`@chat` / `engines/chat/`) exclusively — no more direct `vc.*` schema queries from app-level DAL files.

## Context

VCSM's chat feature has a dual-path architecture:
1. **Primary path (keep):** Engine hooks/controllers via `@chat` → `chat.*` schema
2. **Legacy/fallback path (remove):** Own DAL files → `vc.*` schema

The legacy DAL files use a different schema with different column names (`message_type` vs `message_kind`, `is_active` vs `membership_status`, `is_group` vs `conversation_kind`, etc.) and must be fully removed so VCSM runs exclusively through the engine.
