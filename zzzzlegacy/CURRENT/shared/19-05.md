## Execution Summary

Full forensic review complete. Logan doc written: `vcsm/chat/vcsm.chat.message-flow-audit.md`.

**Top critical findings:**
1. **Full thread re-render on every message** — `MessageList` uses index-based group keys + no `React.memo` on `MessageGroup`/`MessageBubble`; 100-message thread rebuilds entirely on each new message
2. **Inbox full refetch on every message** — `useInbox` subscription fires `load()` (all entries) on any change across any conversation
3. **`__optimistic` flag generated but never used** — send feedback is silently missing; user can't tell if message is pending
4. **Auto-scroll fires unconditionally** — user reading old messages gets jumped to bottom on every new arrival
5. **Failed message vanishes** — no persistent failure state; user must re-type lost content
6. **Silent `markConversationRead` failure** — swallowed error means unread badges can desync silently
7. **Race condition** on initial load + realtime subscription for cross-user messages
